设计循环队列

发布于 2022-04-27 | 分类于 队列 、 算法 | 4分钟 | 802字数

题目简介

设计你的循环队列实现。 循环队列是一种线性数据结构，其操作表现基于 FIFO（先进先出）原则并且队尾被连接在队首之后以形成一个循环。它也被称为“环形缓冲器”。

循环队列的一个好处是我们可以利用这个队列之前用过的空间。在一个普通队列里，一旦一个队列满了，我们就不能插入下一个元素，即使在队列前面仍有空间。但是使用循环队列，我们能使用这些空间去存储新的值。

你的实现应该支持如下操作：

MyCircularQueue(k): 构造器，设置队列长度为 k 。
Front: 从队首获取元素。如果队列为空，返回 -1 。
Rear: 获取队尾元素。如果队列为空，返回 -1 。
enQueue(value): 向循环队列插入一个元素。如果成功插入则返回真。
deQueue(): 从循环队列中删除一个元素。如果成功删除则返回真。
isEmpty(): 检查循环队列是否为空。
isFull(): 检查循环队列是否已满。

示例：

MyCircularQueue circularQueue = new MyCircularQueue(3); // 设置长度为 3
circularQueue.enQueue(1);  // 返回 true
circularQueue.enQueue(2);  // 返回 true
circularQueue.enQueue(3);  // 返回 true
circularQueue.enQueue(4);  // 返回 false，队列已满
circularQueue.Rear();  // 返回 3
circularQueue.isFull();  // 返回 true
circularQueue.deQueue();  // 返回 true
circularQueue.enQueue(4);  // 返回 true
circularQueue.Rear();  // 返回 4

题目分析

因为是一个循环队列，所以如上图所示，我们每次都要算出来头尾节点所在的索引值。对于一个循环队列而言，任何地方可以是head，那么要根据长度跟当前存储的元素个数，计算出tail的index。
公示: tailIdex = （headIndex+count-1）% capacity

java代码实现

class MyCircularQueue {
    //循环队列
    private final int[] queue;
    //头节点
    private int headIndex;
    //尾节点
    private int tailIndex;
    //当前队列中有多少个元素
    private int count;

    public MyCircularQueue(int k) {
        this.queue = new int[k];
        this.headIndex = 0;
        this.tailIndex = 0;
        this.count = 0;
    }

    /**
     * 入队
     *
     * @param value 入队的值
     * @return 是否入队成功
     */
    public boolean enQueue(int value) {
        if (isFull()) {
            return false;
        }
        this.count++;
        tailIndex = (headIndex + count - 1) % queue.length;
        queue[tailIndex] = value;
        return true;
    }

    /**
     * 出队列
     *
     * @return 是否出队成功
     */
    public boolean deQueue() {
        if (isEmpty()) {
            return false;
        }
        this.count--;
        if (headIndex == queue.length - 1) {
            headIndex = 0;
        } else {
            headIndex++;
        }
        return true;
    }

    /**
     * 获取队头的元素
     *
     * @return 队头元素
     */
    public int Front() {
        if (isEmpty()) {
            return -1;
        }
        return queue[headIndex];
    }

    /**
     * 获取队尾的元素
     *
     * @return 队尾元素
     */
    public int Rear() {
        if (isEmpty()) {
            return -1;
        }
        return queue[tailIndex];
    }

    /**
     * 当拥有的元素为0代表是空的
     *
     * @return 是否为空
     */
    public boolean isEmpty() {
        return count == 0;
    }

    /**
     * 存的元素跟queue的长度一致就满了
     *
     * @return 是否满了
     */
    public boolean isFull() {
        return count == queue.length;
    }
}

go代码实现

type MyCircularQueue struct {
	array                       []int
	headIndex, tailIndex, count int
}

func Constructor(k int) MyCircularQueue {
	return MyCircularQueue{
		array:     make([]int, k, k),
		headIndex: 0,
		tailIndex: 0,
		count:     0,
	}
}

func (this *MyCircularQueue) EnQueue(value int) bool {
	if this.IsFull() {
		return false
	}
	this.count++
	this.tailIndex = (this.headIndex + this.count - 1) % len(this.array)
	this.array[this.tailIndex] = value
	return true
}

func (this *MyCircularQueue) DeQueue() bool {
	if this.IsEmpty() {
		return false
	}
	if this.headIndex == len(this.array)-1 {
		this.headIndex = 0
	} else {
		this.headIndex++
	}
	this.count--
	return true
}

func (this *MyCircularQueue) Front() int {
	if this.IsEmpty() {
		return -1
	}
	return this.array[this.headIndex]
}

func (this *MyCircularQueue) Rear() int {
	if this.IsEmpty() {
		return -1
	}
	return this.array[this.tailIndex]
}

func (this *MyCircularQueue) IsEmpty() bool {
	return this.count == 0
}

func (this *MyCircularQueue) IsFull() bool {
	return this.count == len(this.array)
}

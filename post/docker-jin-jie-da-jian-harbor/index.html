Docker进阶-搭建Harbor

发布于 2020-04-22 | 分类于 Docker | 3分钟 | 522字数

Harbor介绍

Harbor is an open source trusted cloud native registry project that stores, signs, and scans content. Harbor extends the open source Docker Distribution by adding the functionalities usually required by users such as security, identity and management. Having a registry closer to the build and run environment can improve the image transfer efficiency. Harbor supports replication of images between registries, and also offers advanced security features such as user management, access control and activity auditing.

Harbor搭建流程

配置HTTPS(没有第三方的证书，自己创建证书可以用在测试与开发环境,测试域名使用harbor.gxy.com)

  • 下载Harbor releases page
  • 解压下载下来的文件夹 tar xvf harbor-offline-installer-v1.10.2.tgz
  • 生成证书颁发机构证书
  1. 生成CA证书的私钥:
openssl genrsa -out ca.key 4096
  1. 生成CA证书:
openssl req -x509 -new -nodes -sha512 -days 3650 -subj "/C=CN/ST=Beijing/L=Beijing/O=example/OU=Personal/CN=harbor.gxy.com" -key ca.key -out ca.crt
  • 生成服务器证书
  1. 生成服务器的私钥
openssl genrsa -out harbor.gxy.com.key 4096
  1. 生成证书签名请求(CSR)
openssl req -sha512 -new  -subj "/C=CN/ST=Beijing/L=Beijing/O=example/OU=Personal/CN=harbor.gxy.com" -key harbor.gxy.com.key  -out harbor.gxy.com.csr
  1. 生成一个x509 v3扩展文件
cat > v3.ext <<-EOF
authorityKeyIdentifier=keyid,issuer
basicConstraints=CA:FALSE
keyUsage = digitalSignature, nonRepudiation, keyEncipherment, dataEncipherment
extendedKeyUsage = serverAuth
subjectAltName = @alt_names
[alt_names]
DNS.1=horbor.gxy.com
EOF
  1. 使用v3.ext文件为您的Harbor主机生成证书
openssl x509 -req -sha512 -days 3650 \
    -extfile v3.ext \
    -CA ca.crt -CAkey ca.key -CAcreateserial \
    -in harbor.gxy.com.csr \
    -out harbor.gxy.com.crt
  • 提供证书给Harbor和Docker
  1. 把服务器证书与服务器私钥移动到harbor下进行管理
cp harbor.gxy.com.crt /usr/local/harbor/cert/
cp harbor.gxy.com.key /usr/local/harbor/cert/
  1. 将harbor.gxy.com.crt转换为harbor.gxy.com.cert，以供Docker使用
openssl x509 -inform PEM -in harbor.gxy.com.crt -out harbor.gxy.com.cert
  1. 服务器的证书，key，ca拷贝到Docker的认证文件夹下
cp harbor.gxy.com.cert /etc/docker/certs.d/harbor.gxy.com/
cp harbor.gxy.com.key /etc/docker/certs.d/harbor.gxy.com/
cp ca.crt /etc/docker/certs.d/harbor.gxy.com/
  1. 重启Docker
systemctl restart docker

  • 配置你的Harbor.yml

    其它部分可按需修改-官方配置文档

  • 启动Harbor

sh ./install.sh --with-notary
  • 配置本地证书
  • 用https访问你的Harbor
